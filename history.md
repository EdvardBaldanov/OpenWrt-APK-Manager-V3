# История разработки OpenWrt APK v3 Manager

В этом файле зафиксированы основные этапы трансформации проекта из базового скрипта в полноценную систему управления репозиторием нового поколения.

## 📅 Март 2026: Глобальное обновление

### 1. Переход на стандарт APK v3
- Внедрена поддержка бинарных индексов `packages.adb`.
- Реализована генерация и управление ключами **P-256** для подписи пакетов.
- Созданы механизмы формирования бинарных (`.sig`) и текстовых (`.asc`) подписей по стандартам OpenWrt 25.12+.

### 2. Рефакторинг архитектуры
- **Разделение конфигурации**: Вместо единого файла настройки разнесены по `auth.json` (GitHub), `repos.json` (список отслеживания), `filters.json` (фильтры) и `web_auth.json` (доступ к панели).
- **Унификация хранилища**: Реализован переход к единой структуре пакетов, что упростило индексацию и подключение к роутерам.
- [18.03.2026] Рефакторинг UI: Переход на вкладочный интерфейс (Обзор, Пакеты, Источники, Настройки). Добавлен живой вывод логов.
- [18.03.2026] Оптимизация: Сокращение размера репозитория (bin/ 144MB -> 8.6MB). Публикация на GitHub.
- [18.03.2026] UX: Интеллектуальный блок подключения к роутеру с авто-подстановкой IP сервера.
- [18.03.2026] Fix: Новая система конфигурации (раздельные JSON) с миграцией и защитой от перезаписи.

### 3. Веб-интерфейс "Next-Gen"
- **Дизайн**: Внедрена современная эстетика Glassmorphism (стеклянный интерфейс) с использованием CSS-переменных и плавных анимаций.
- **Интерактивность**:
    - Добавлен **File Browser** для удобного просмотра списка скачанных пакетов.
    - Реализован **Settings Editor**, позволяющий менять настройки сервера и список репозиториев прямо в браузере.
- **Локализация**: Шрифты заменены на **Inter** и **Montserrat** с полной поддержкой кириллицы.

### 4. Автоматизация и UX
- **Unified Workflow**: Кнопки синхронизации и сборки объединены в один процесс "Обновить всё".
- **Планировщик (Cron)**: Добавлен фоновый поток, автоматически проверяющий обновления каждые N часов.
- **Real-time Мониторинг**: 
    - Реализовано модальное окно логов.
    - Добавлен **постоянный раздел системных логов** на главной странице для мгновенного контроля состояния.

### 5. Безопасность
- Внедрена **Basic Auth** аутентификация для защиты панели управления.
- Реализовано скрытое хранение чувствительных данных (GitHub Token).
