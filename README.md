# OpenWrt APK v3 Manager (Next Generation)

Современный менеджер репозиториев для OpenWrt, построенный с нуля на базе опыта работы с форматом APK v3.

## Структура проекта
- `main.py` - Точка входа.
- `app/` - Основной код приложения.
- `data/` - Папка для данных пользователя (конфиги, ключи, пакеты).
- `bin/` - Внешние инструменты (apk-tools).

## Особенности
- Поддержка подписей P-256 (APK v3).
- Модульная архитектура.
- Современный веб-интерфейс.

## Разработка
Для добавления новых функций используйте структуру в `app/services/` для бизнес-логики и `app/core/` для низкоуровневых операций с пакетами.
